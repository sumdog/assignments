The goal of this assignment is to compare the Java built-in SAX parser as compared to the xerces DOM parser to see which one performed better. For this experiment I modified two of the previous java assignments designed to parse a single Shakespeare play in XML format for a word or phrase. The modified programs perform as the originals, however they have the addition of outputting their execution times to standard error (System.err in Java). The program writes a single line to stderr in the following format:

<file name>,<execution time for SAX>,<execution time for DOM>

The test computer was a MacBook running Mac OS X 10.4.8 and version 1.5.0_06-112 of the Java Runtime. It has a 2.0Ghz Intel processor and 2GB of Ram
