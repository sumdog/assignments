9.10 Consider a demand-paging system with the following time-measured utilizations:

  1. CPU utilization 20%
  2. Paging disk 97.7%
  3. Other I/O devices 5%
Which (if any) of the following will (probably) improve CPU utilization?
Explain your answer.
  1. Install a faster CPU.
  2. Install a bigger paging disk.
  3. Increase the degree of multiprogramming.
  4. Decrease the degree of multiprogramming.
  5. Install more main memory.
  6. Install a faster hard disk or multiple controllers with multiple hard disks.
  7. Add prepaging to the page fetch algorithms.
  8. Increase the page size.

The CPU isn't even at 1/4 of full utilization so a faster one will not increase performance. The swap disk is at maximum capacity so increasing its size will increase performance. Increasing the degree of parallel programming will help only if the system has multiple CPUs. Decreasing the about of parallelism will help if the system has a single CPI. More main memory would help by alleviating some of the burden carried by the page file. The Disk is only being utilized 5% of the time, so faster hard drives/controllers will not significantly affect performance.
