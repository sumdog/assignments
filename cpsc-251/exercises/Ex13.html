11.7 Fragmentation on a storage device could be eliminated by recompaction of the information.  Typical disk devices do not have relocation or base registers (such as are used when memory is to be compacted), so how can we relocate files? Give three reasons why recompacting and reallocation of files are often avoided.

It is typically too time consuming and requires too much overhead to relocate files on a storage device, but you can do so by collecting all of the files on the disk and then reallocate each file back to the disk in an order that compacts the files together.  However this is generally avoided because it takes entirely too much time, it generally requires all or most of the resources from a system so no other tasks can be carried out or performance can greatly suffer, and sometimes the process can cause more problems in the system because it can be hard to estimate the size of a file and cause it to be allocated to a space that is either too big or too small.

11.8 In what situations would using memory as a RAM disk be more useful than using it as a disk cache.

Using memory as a RAM disk would be more useful when you are only working with a few small files that you want to access and manipulate as quickly as possible.  A RAM disk does not allow for large storage, and is volatile memory.

11.10 Explain why logging metadata updates ensures recovery of a file system after a file-system crash.

Logging metadata updates ensures recovery of a file system after a crash because a set of operations for performing a specific task are written to a log which is either in a separate section of a file system or on a separate disk.  As changes are made a pointer is updated to indicate which actions have completed and which are still incomplete.  The log file is a circular buffer which writes over old values when it runs out of space.  Therefore if a system crashes the log file will only contain speficic tasks that were incomplete, after the crash the tasks in the log can then be completed so the file system remains constant.
