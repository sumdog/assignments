  3. What Linux command do you use to list detailed information about the files in your directory?
    • ls -la will give a listing of all files, including hidden files, with all of the files information
  4. How would you create a file named Test by typing the line abc at the keyboard?
    • You would type echo "abc" > Test
  5. What commands are used with directories? What does each of these commands do?
    • mkdir - makes a directory
    • rmdir - removes a directory if it is empty
    • ls - list the files in a directory
    • rm -rf - removes a non-empty directory
  6. How to you copy files? Delete files?
    • cp - copies files
    • mv - moves files
    • rm - deletes files
  7. How do you create filename aliases? Give examples to illustrate your answer.
    • The ln command creates links. A link can either be symbolic, which creates a file that points to another file, or it can be hard, where two files point to the same location on disk.
